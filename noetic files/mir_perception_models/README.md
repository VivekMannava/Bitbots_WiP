## mir_perception_models

This repository contains perception models for industrial robotics.

Directory structure:

```
├── CMakeLists.txt
└── package.xml
├── common
    └── models
        ├── <model_name>
            └── <dataset>
                ├── model.ckpt.data-00000-of-00001
                ├── model.ckpt.index
                └── model.ckpt.meta
```
